# MIDI Gen

AI-powered music producer in the browser. Describe what you want to hear and MIDI Gen creates multi-layer compositions with synthesizers, drums, and effects.

## Features

- Natural language music creation via AI
- Multi-layer synth engine with 12+ built-in presets
- Custom patch design (oscillators, filters, envelopes, effects)
- Drum machine with pattern sequencing
- Song persistence (save/load/export/import)
- Section-based song structure (8/16/32/64 bar sections)
- Effects chain: reverb, delay, chorus, distortion, phaser, and more

## Usage

Open `index.html` in a browser. Type a music description in the chat and let the AI compose for you.
